UrMentor — Dynamic ML Roadmap System
Overview

UrMentor is designed as a personalized ML learning companion, not just a static roadmap. Instead of giving the same fixed path to everyone, it adapts to each user based on their goals, background, and progress.

The goal is to make the ML journey structured, engaging, and consistent, while reducing confusion about what to learn, how to learn, and what to prioritize .

Core Idea: Dynamic Roadmap Generation

The roadmap is not pre-defined.

Instead:

The system first asks the user a few questions, such as:
Current skill level
Target role (ML Engineer, Data Scientist, etc.)
Time commitment
Interests
Based on these inputs, a custom roadmap is generated dynamically.
This ensures:
No unnecessary topics
Focus only on what matters
Personalized learning journey
Learning Flow Structure
1. Introductory Slides (Layman Explanation)

Before starting the roadmap, the platform provides:

A simple slideshow
Explaining What Machine Learning is
In very easy, non-technical language

This ensures even beginners can start confidently.

2. Roadmap Execution

The roadmap is divided into learning tasks, where each task contains:

📺 YouTube videos
📄 Research papers / blogs
📊 Datasets (if applicable)

After completing each task:

👉 The user must take a short quiz
→ This ensures actual understanding, not passive learning

3. Progress + Consistency System

UrMentor focuses heavily on consistency, not just content.

Features include:

🎯 Weekly target setting (auto-generated or user-defined)
🔔 Notifications & reminders
📈 Progress tracking

This helps users:

Stay disciplined
Avoid burnout
Maintain steady growth
Role of Gemini API

The entire system is powered using the Gemini API, which acts as the core intelligence behind the platform.

Gemini is used for:
🧠 Generating personalized roadmaps
📝 Creating quizzes dynamically
🎯 Adjusting weekly targets
🔄 Adapting roadmap based on progress
💬 Acting as the coding/learning assistant

In short, Gemini functions as the brain of UrMentor, enabling real-time personalization and intelligent guidance.

Objective Summary

UrMentor aims to:

Replace static roadmaps with adaptive learning paths
Make ML learning interactive and engaging
Ensure consistency through tracking and notifications
Provide a guided, structured journey from beginner to advanced level
