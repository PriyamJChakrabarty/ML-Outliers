# ML Outliers — Roadmap Resources

## Introduction

The overall roadmap goes as follows — start with Python basics, then the core libraries (NumPy, Pandas, Matplotlib, Seaborn), then traditional ML algorithms, and finally Deep Learning concepts. This builds a strong foundation to explore AI/ML through projects.
